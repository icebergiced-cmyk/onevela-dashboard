หนังสือจองสิทธิ์ (ใบจอง)
โครงการวันเวลา อมตะนคร ชลบุรี
ทำที่ สำนักงานขายโครงการวันเวลา เลขที่ใบจอง - วันที่ -

ข้าพเจ้า - (“ผู้จอง”) (เรียกว่าผู้ซื้อ) มีที่อยู่ตามทะเบียนบ้านเลขที่ - เลขที่บัตรประชาชน - โทรศัพท์ - มีความประสงค์ขอจองบ้านในโครงการวันเวลาอมตะนคร โดยบริษัท ทู บิลด์ ดีเวลลอปเม้นท์ จำกัด ซึ่งต่อไปจะเรียกว่า “โครงการ” ที่ตั้งบริษัท เลขที่ 89 ตำบลคลองตำหรุ อำเภอเมือง จังหวัดชลบุรี โทรศัพท์ 092-787-4222 ซึ่งต่อไปนี้จะเรียกว่าผู้จะขาย

จองบ้านเลขที่ - แปลง - แบบ - - พื้นที่ประมาณ - ตารางวา ในราคา 0 บาท (-)

1. ในวันนี้ผู้จองได้ชำระเงินจองเป็นเงิน 0 บาท (-) ให้แก่โครงการไว้แล้ว โดยชำระเป็น

เงินสด
โอนบัญชี ธนาคารไทยพาณิชย์ เลขที่บัญชี 409-691797-6 ชื่อบัญชี บริษัท ทูบิลด์ ดีเวลลอปเม้นท์ จำกัด ลงวันที่ -
เช็คธนาคาร ลงวันที่ .................

กรณีชำระเงินด้วยเช็ค หากเช็คเรียกเก็บเงินไม่ได้ให้ถือว่าผู้จองสละสิทธิ์ในการจองบ้านตามใบจองฉบับนี้ทันที และให้ถือว่ากำหนดวันที่จะทำสัญญาจะซื้อจะขายตามวรรคแรกเป็นอันยกเลิกไม่มีผลผูกพันต่อโครงการอีกต่อไป และยินยอมให้โครงการนำบ้านตามที่ระบุในใบจองนี้ออกขายให้กับบุคคลอื่นได้ทันทีโดยไม่จำเป็นต้องบอกกล่าวแก่ผู้จองแต่อย่างใด

2. ผู้จองและผู้รับจองตกลงจะทำสัญญาจะซื้อจะขายที่ดินพร้อมสิ่งปลูกสร้างที่จองตามสัญญาข้อ 1. กำหนดภายใน - วัน นับแต่วันทำหนังสือจองฉบับนี้ หรือภายในวันที่ - โดยจะชำระค่างวด ณ วันทำสัญญาจะซื้อจะขายเป็นจำนวนเงิน 0 บาท (-) และชำระค่างวดเป็นจำนวนเงินดังนี้

งวดที่ 1 ถึง งวดที่ - งวดละ 0 บาท (-) ชำระภายในวันที่ - ของทุกเดือน เริ่มตั้งแต่เดือน - ถึงเดือน -

รวมยอดผ่อนดาวน์ทั้งสิ้นจำนวน 0 บาท (-)

คงเหลือยอดเงินชำระ ณ วันโอนกรรมสิทธิ์ จำนวน 0 บาท (-)

3. หากผู้จองไม่ทำสัญญาจะซื้อจะขายภายในกำหนดเวลาข้อ 1. ถือว่าผู้จองสละสิทธิ์ในการจองบ้านตามใบจองฉบับนี้ ยินยอมให้โครงการริบเงินทั้งหมดโดยไม่เรียกสิทธิใด ๆ ขึ้นโต้แย้งทั้งสิ้น และยินยอมให้โครงการนำบ้านที่ระบุในใบจองนี้ออกขายให้กับบุคคลอื่นได้ทันที โดยไม่จำต้องบอกกล่าวแก่ผู้จองแต่อย่างใด

4. ผู้จองจะโอนสิทธิตามใบจองฉบับนี้ให้แก่บุคคลอื่นไม่ได้ เว้นแต่จะได้รับความยินยอมเป็นหนังสือจากโครงการก่อน และเมื่อโครงการให้ความยินยอมแล้ว ผู้จองตกลงเสียค่าใช้จ่ายในการโอนสิทธิให้แก่โครงการเป็นเงิน 25,000 บาท (สองหมื่นห้าพันบาทถ้วน) ทั้งนี้ผู้รับโอนสิทธิจะต้องรับไปซึ่งสิทธิหน้าที่และความรับผิดของผู้จองตามใบจองฉบับนี้ทุกประการ

หนังสือจองสิทธิ์นี้ทำขึ้นเป็นสองฉบับ มีข้อความถูกต้องตรงกัน คู่สัญญาได้อ่านและเข้าใจข้อความ โดยตลอดดีแล้ว จึงลงลายมือชื่อพร้อมทั้งประทับตรา (ถ้ามี) ไว้เป็นสำคัญต่อหน้าพยาน และเก็บไว้ฝ่ายละหนึ่งฉบับ
