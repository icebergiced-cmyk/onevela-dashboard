## สำหรับ iPhone / iPad (Safari)

1. เปิด **Safari** (สำคัญ — ต้องเป็น Safari เท่านั้น ไม่ใช่ Chrome)
2. พิมพ์ URL Dashboard
3. กดไอคอน **"แชร์"** ที่กลางจอด้านล่าง (รูปกล่องลูกศรชี้ขึ้น)
4. เลื่อนลง เลือก **"Add to Home Screen"** (เพิ่มไปยังหน้าจอโฮม)
5. กดปุ่ม **"Add"** (เพิ่ม) มุมขวาบน
6. กลับไปหน้าจอโฮมของ iPhone จะเห็นไอคอน **V** สีส้ม-น้ำเงิน
7. กดไอคอน → Dashboard เปิดเต็มจอเหมือนแอปจริง

---

## สำหรับ Android (Chrome)

1. เปิด **Chrome**
2. พิมพ์ URL Dashboard
3. กดเมนู **3 จุด** มุมขวาบน
4. เลือก **"Add to Home screen"** (เพิ่มลงในหน้าจอหลัก)
5. ถ้ามี popup ขึ้นถาม กด **"Install"** หรือ **"Add"**
6. กลับไปหน้าจอโฮม จะเห็นไอคอน Dashboard
7. กดไอคอน → เปิดเหมือนแอปจริง

> Samsung Internet Browser ก็ใช้ได้ — กดเมนู 3 ขีด → "Add page to" → "Home screen"

## ฟีเจอร์ที่ใช้ได้บนแอป

- ดู Dashboard ทุกหน้า (Master + Detail รายแปลง)
- กรองตามไทป์, สถานะ, ค้นหา
- ดูได้ทุกที่ทุกเวลา (มีสัญญาณ)
- เปิดได้แม้สัญญาณอ่อน (cache ช่วย)

## ข้อจำกัดที่ต้องเข้าใจ

- **อ่านอย่างเดียว** — ยังแก้ข้อมูลผ่านแอปไม่ได้ (จะเปิดให้แก้ในเฟส 2)
- **ข้อมูลอัพเดทเมื่อผู้ดูแลแก้ไฟล์** — ไม่ใช่ real-time
- ดูเวลาอัพเดทล่าสุดได้ที่ส่วน "อัปเดตข้อมูล" บนหัว Dashboard

## ปัญหาที่อาจเจอ

**Dashboard เปิดช้า ครั้งแรก**

ครั้งแรก browser ต้องดาวน์โหลดทั้งหมด (ประมาณ 600KB) ใช้เวลา 2-5 วินาทีบนสัญญาณ 4G ครั้งต่อๆ ไปจะเร็วมาก เพราะ cache แล้ว

**ข้อมูลไม่อัพเดท**

ปิดและเปิดแอปใหม่ ถ้ายังไม่อัพเดท เปิดผ่าน browser แล้วกด refresh (ดึงลง)

**ไอคอน "V" ไม่ขึ้น เห็นเป็นรูปอื่น**

iPhone บางรุ่นต้องเปิดผ่าน Safari ก่อนเท่านั้น ห้ามใช้ Chrome iOS

**ลืม URL**

เปิด Notes บันทึก URL ไว้ก่อนติดตั้ง หรือผู้ดูแลส่ง URL ทาง LINE กลุ่ม
