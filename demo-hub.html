🎯 ลองเล่นตาม Workflow นี้

1
เปิด หน้า Admin admin → กด "+ เริ่มก่อสร้างแปลงใหม่" → ค้น "230" → เลือกแปลง → กดเริ่ม
2
เปิด หน้าโฟร์แมน foreman → login โก้ (หรือใหญ่) → แปลงใหม่ที่เพิ่งเริ่มจะอยู่ในรายการ
3
เลือกแปลงนั้น → เลือก milestone ที่เสร็จ → ถ่ายรูป (mock) → ใส่บันทึก → กด "บันทึกอัปเดต"
4
กลับมา หน้า Admin → เห็น % progress + อัปเดตล่าสุดของแปลงนั้นเปลี่ยนทันที
5
เปิด หน้า Tracker tracker → เลือกแปลงเดียวกัน → เห็นบันทึกที่โฟร์แมนเพิ่งใส่ + รูปที่เพิ่งถ่าย
6
กลับ Admin → กดปุ่ม "หยุด" → แปลงนั้นจะหายจาก dropdown โฟร์แมน
