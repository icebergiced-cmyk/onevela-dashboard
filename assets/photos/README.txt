📸 รูปบ้านโครงการวันเวลา (One Vela)

วิธีอัพรูปจริง:

1. รูปต้องเป็น .jpg หรือ .png (แนะนำ .jpg, max 500KB/ไฟล์)
2. ตั้งชื่อตามรูปแบบ: <type>_<view>.jpg
   • type = A, B, C, D
   • view = exterior, floorplan, interior, siteplan
3. วางไฟล์ในโฟลเดอร์นี้ (assets/photos/)
4. Dashboard จะใช้ .jpg ก่อน .svg อัตโนมัติ
5. รัน script รวมแล้ว deploy ใหม่

ตัวอย่างไฟล์ที่ต้องมี (16 ไฟล์):
  A_exterior.jpg, A_floorplan.jpg, A_interior.jpg, A_siteplan.jpg
  B_exterior.jpg, B_floorplan.jpg, B_interior.jpg, B_siteplan.jpg
  C_exterior.jpg, C_floorplan.jpg, C_interior.jpg, C_siteplan.jpg
  D_exterior.jpg, D_floorplan.jpg, D_interior.jpg, D_siteplan.jpg
