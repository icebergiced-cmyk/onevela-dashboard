
## สิ่งที่แก้ได้ (ช่องสีเหลือง)

| คอลัมน์ | แก้เมื่อไหร่ |
|---------|------------|
| **สถานะการขาย** | ลูกค้าจอง / ทำสัญญา / โอน / ยกเลิก |
| **ราคาขายสุทธิ** | มีส่วนลดพิเศษ หรือเปลี่ยนแผนการขาย |
| **บ้านเลขที่** | กรมที่ดินกำหนดเลขใหม่ |
| **หมายเหตุ** | บันทึกพิเศษเรื่องลูกค้า |
| **เลขโฉนด** | ได้รับโฉนดใหม่ |
| **เนื้อที่** | กรมที่ดินวัดใหม่ |

### สถานะการขาย — เลือกจาก dropdown ได้ 5 ค่า

- **ว่าง** — ยังไม่มีลูกค้า
- **จอง** — ลูกค้าวางเงินจอง
- **สัญญา** — เซ็นสัญญาซื้อขายแล้ว
- **ขายดาวน์** — ลูกค้าจ่ายดาวน์ครบ
- **โอนแล้ว** — โอนกรรมสิทธิ์ที่กรมที่ดินแล้ว

## สิ่งที่ห้ามทำ ❌

1. **ห้ามลบแถว** — Dashboard ต้องการ 483 แปลงครบทุกแถวเสมอ
2. **ห้ามแก้แถวที่ 1** (ภาษาอังกฤษ: plot, typeKey, ...) — Dashboard อ่านจากแถวนี้
3. **ห้ามแก้ช่องสีเทา** — เป็น identifier ของระบบ (แปลง, ไทป์, ไทป์เต็ม, รหัสไทป์)
4. **ห้ามเพิ่มคอลัมน์ใหม่** — Dashboard ไม่รู้จัก
5. **ห้ามแก้ชื่อคอลัมน์**

---

## ขั้นตอนการแก้ข้อมูล

### กรณีที่ 1: ลูกค้าทำสัญญาแปลง 5

1. เปิด Google Sheet
2. ไปที่แท็บ **metadata**
3. หาแถวของแปลง 5 (เลื่อนลง 5 แถว หรือ Ctrl+F หา "5")
4. คลิกที่ช่อง "สถานะการขาย"
5. กดลูกศรลง → เลือก **"สัญญา"**
6. (ถ้ามี) ไปคอลัมน์หมายเหตุ พิมพ์ "ทำสัญญา 18 พ.ค. 69 ลูกค้า: คุณสมชาย"
7. เสร็จ — รอ 5 นาที Dashboard อัพเดท

### กรณีที่ 2: โอนแปลง 12 ราคาสุทธิ 4.1 ล้าน

1. หาแถวของแปลง 12
2. สถานะการขาย → **"โอนแล้ว"**
3. ราคาขายสุทธิ → **4100000** (ใส่ตัวเลขไม่ต้องมีจุลภาค)
4. หมายเหตุ → "โอน 18 พ.ค. 69"

### กรณีที่ 3: ได้บ้านเลขที่ใหม่จากกรมที่ดิน

1. หาแถวของแปลงนั้น
2. บ้านเลขที่ → ใส่เลขใหม่ เช่น **"89/27"**

## ตรวจว่า Dashboard อัพเดทแล้ว

หลังแก้เสร็จ:

1. เปิด Dashboard บนมือถือ / browser
2. ดูที่ **หัว Dashboard** ใต้คำว่า "อัปเดตข้อมูล..." จะเห็นบรรทัด:
   - 🟢 **ซิงค์เมื่อ HH:MM — 483 แปลง (N เปลี่ยน)**
3. ถ้าอยากเห็นทันทีไม่รอ 5 นาที กด **"รีเฟรช"** ที่บรรทัดนั้น

## คำถามที่พบบ่อย

**Q: แก้แล้วต้องกดบันทึกไหม?**
A: ไม่ต้อง Google Sheet auto-save ทุกครั้ง — แค่กด Tab หรือคลิกที่อื่นก็ saved

**Q: ทีม Sales เปิดพร้อมกันได้ไหม?**
A: ได้ — Google Sheet รองรับ multi-user real-time

**Q: แก้ผิดแก้คืนได้ไหม?**
A: ได้ กด Ctrl+Z (Mac: Cmd+Z) หรือดู History: File → Version history → See version history

**Q: ใครเห็นว่าใครแก้อะไรบ้าง?**
A: คลิกขวาที่ cell → "Show edit history" จะเห็นว่าใครแก้อะไรเมื่อไหร่

**Q: เปิดบนมือถือได้ไหม?**
A: ได้ Google Sheets app บน iOS/Android แก้ได้สะดวกพอสมควร

**Q: Dashboard ไม่อัพเดทเลย ทำยังไง?**
A: ที่ Dashboard กด "รีเฟรช" ที่หัว ถ้ายังไม่ขึ้นเปิด-ปิดแอป browser ใหม่ ถ้ายังไม่ได้แจ้งผู้ดูแล
